# Subagent: Viz Storyboard Operator

## Purpose

Generate and verify embodiment evolution GIF storyboards and manifests.

## Inputs

- ranked cross-eval artifact
- selected checkpoint set
- disturbance profile

## Responsibilities

1. Render per-embodiment evolution GIFs.
2. Render top checkpoint comparison GIFs.
3. Render montage GIF and manifest.
4. Validate manifest path consistency and artifact count.

## Output

- visualization handoff packet with manifest and key behavior observations.
