# Lessons Taxonomy

Use tags in handoff packets and lesson references for selective retrieval.

## Core Tags

- `common`: broadly reusable pattern.
- `unique`: edge scenario with narrower applicability.
- `failure-mode`: known failure pattern and mitigation.
- `hardware`: CUDA/device/runtime portability behavior.
- `convergence`: ratchet/gating/promotion behaviors.
- `viz`: storyboard/visual diagnostics behavior.
- `handoff`: continuity/transfer process pattern.
